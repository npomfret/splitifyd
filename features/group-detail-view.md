# Group Detail View

**Status:** Implemented

## Description
The group detail view provides a comprehensive overview of a specific group, including its expenses, balances, and management options.

## Implemented Items
- Group balance summary (who owes whom)
- Group expenses list
- Add expense button (pre-populated with this group)
- Group settings/members management
- Simplify debts view for this group
- Group activity history
